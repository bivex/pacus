Декларация доступности (ISO 9241-171)

Дата: 01.05.2026

Обязательство

Система «pacus» стремится обеспечить доступность для всех пользователей, включая людей с ограниченными возможностями. Мы ориентируемся на требования ISO 9241-171:2008 и WCAG 2.1 уровня AA.

Что реализовано

  • Язык документа: Все страницы содержат атрибут lang="ru"
  • Навигация с клавиатуры: Все интерактивные элементы доступны через Tab/Shift+Tab
  • Визуальная индикация фокуса: Стилизованные контуры (focus-visible) для ссылок и кнопок
  • Semantic HTML: Использование <nav>, <main>, <table> с scope, aria-label
  • Screen reader поддержка: Класс .sr-only для визуально скрытого, но доступного контента
  • Статусы не только цветом: Текстовые метки статусов дублируют цветовую индикацию
  • Адаптивная вёрстка: viewport meta-тег, grid-layout
  • Печать: Оптимизированные стили для печати через @media print
  • Контрастность: Основной текст #111827 на фоне #fff (контраст > 15:1)
  • Поддержка без JavaScript: Элемент <noscript> с уведомлением, основной контент доступен
